Pengaturan QC

Panduan Aplikasi

1. Menu Input QC

  • Pilih Kategori Produk di bagian atas layar.
  • Ketik/Pilih Range Serial Number dan Nama Project.
  • Scan atau ketik Serial Number (SN) unit. Wajib diisi sebelum buka kamera!
  • Klik tombol Buka Kamera. Bukti foto akan otomatis tertempel Watermark (Project, Range SN, Waktu & Lokasi).
  • Klik Upload File. Sistem akan otomatis membuat folder di Drive sesuai data yang diisi.

2. Menu Database

  • Gunakan menu ini untuk mencari file RCFGX dan Firmware.
  • Navigasi bekerja seperti Folder di Komputer. Klik folder untuk masuk, tekan tombol Kembali untuk keluar.

3. Menu Laporan (PDF)

  • Gunakan menu ini saat proses foto QC telah selesai.
  • Ketik Nama Folder Target tempat foto disimpan (Biasanya Range SN / Project).
  • Isi Nama Project dan Range SN untuk ditampilkan di Dokumen.
  • Klik Bikin Laporan Sekarang!. Sistem meracik foto menjadi PDF.
